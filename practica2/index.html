Página Web

Iker López Bonilla
Ing. Mecatrónica
Oscar Emanuelle Garrido Javalera
Ing. Mecatrónica
Fecha: Agosto 23, 2024
html Electrónica

- Resumen -

En esta práctica, diseñamos una pequeña página web para conocer la programación en html.

La práctica fue realizada en una sola clase, con la orientación de nuestro profesor Huber Girón Nieto.

- Introducción -

Nuestro profesor nos mostró la aplicación Visual Studio Code, la cual sirve para programar páginas web. Al abrir la aplicación, nos dió una guía de las herramientas más utilizadas al crear una página web.

La asignación fue diseñar una pequeña página web que tuviera la mayoría de elementos básicos.

- Materiales -

Unicamente se requirió una computadora con el programa Visual Studio Code.

- Desarrollo -

Después de empezar un código al crear carpetas para cada archivo, insertamos las fórmulas adecuadas a laos códigos para poder agregar títulos, encabezados, colores, tamaños, fuentes, enlaces, imágenes, documentos, cambios de página, entre otros.

- Simulación

Para poder visualizar el progreso, nuestro profesor nos mostró la extensión Live Server, la cual sirve para visualizar el el código que se ha escrito por el momento.

Las páginas podían observarse con claridad, pues habían sido programadas correctamente.

Diseño

A pesar de que las páginas contaban con un diseño sencillo, no resultaban confusas ni distractoras.

- Resultados -

Finalmente, nuestro profesor nos enseño Github; una página capaz de almacenar nuestro programa para poder presentarlo como una página web pública. Simplemente fue necesario crear una cuenta para poder compartir todas las carpetas usadas en la elaboración del código.

Las páginas se guardaron correctamente.

- Conclusiones -

Se logró crear una página web con varios elemtnos capaz de ser publicada globalmente.

Igualmente, comparamos las diferencias entre nuestras dos páginas para encontrar errores y diferencias entre ellas.
